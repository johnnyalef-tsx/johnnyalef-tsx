## 🏛️ Filosofia de Engenharia
Atuo sob um sistema de governação focado em **Soberania Técnica** e **Escalabilidade**. O meu processo de desenvolvimento é guiado por normas rígidas de arquitetura que garantem softwares leves, inquebráveis e otimizados.

### 🛠️ Pilares do meu Código:
- **Modularidade Extrema (Constraint 50):** Código limpo e de fácil manutenção.
- **Soberania de Servidor (RSC):** Foco total em performance bruta (Zero JS no cliente).
- **Arquitetura Matemática:** Interfaces fluidas e precisas via cálculos de `clamp()`.
- **Governação de Estado:** Desenvolvimento ancorado em registos de estado para evitar a entropia.
